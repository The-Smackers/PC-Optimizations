PC Optimization Guide


Windows 11 Preparation

Nothing special/different in Windows 11 so same practices before still apply like below tips, A clean install is always better than updating to Windows 11

Backup all your data always and enable System Restore and create a restore point after a fresh install or before completing this guide

  • Check motherboard website for BIOS updates and apply a fairly recent stable one
  • Check in BIOS utility if XMP settings are (still) applied after update to Windows 11
  • Your Windows drive must be formatted GPT not MBR, yours probably is good but this video shows you how to check


Windows 11 Checklist

  1. Install and update Windows 11
  2. Make sure Game Mode is enabled
  3. Make sure hardware accelerated gpu scheduling and game optimizations is enabled
  4. Power Options set to High Performance or Ultimate Performance
  5. Check Nvidia control panel that global refresh rate and power is set to maximum/high performance
  6. Open task manager and set start up apps to disabled unless actually want them starting with Windows


How to Install/Update NVIDIA GPU Drivers Correctly

Display Driver Uninstaller Setup (to cleanup any currently installed drivers)

  • Download and install DDU and start the program
  • Copy the Options like the image below and then click Close (If you want GeForce Experience uncheck the box)

  • Select your device type and choose NVIDIA or AMD GPU from the dropdown
  • Click Clean and restart
  • Your PC will restart after a couple of minutes be patient


Clean Install of NVIDIA Driver using NVCleanstall

  • Download, install and open NVCleanstall
  • Manually select a driver version and choose the latest 64 bit Desktop driver
  • Click Next and click Next again
  • Copy the settings like the two images below on the Installation Tweaks screen then click Next
  • Click Next and then click Install and wait for the installation to complete
